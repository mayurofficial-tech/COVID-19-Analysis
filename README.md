# 🦠 COVID-19 Data Analysis (EDA Project)

## 📌 Description
This project presents a complete Exploratory Data Analysis (EDA) of the COVID-19 pandemic using real-world data. It analyzes global and regional trends in cases, deaths, and recoveries from 2020 to 2024 using Python and data visualization techniques.

## 🎯 Objective
The main objective of this project is to study COVID-19 trends, compare global and regional impacts, and understand patterns in cases, deaths, and recovery rates across different countries and continents.

## 📊 Dataset
- Source: Our World in Data COVID-19 Dataset  
- Time Period: 2020-01-05 to 2024-08-04  
- Features Used: location, continent, date, total_cases, new_cases, total_deaths, new_deaths, population  

## 🚀 Key Features
- Data Cleaning and Preprocessing  
- Handling Missing Values  
- Exploratory Data Analysis (EDA)  
- Global and Continent-wise Analysis  
- Country-wise Comparison  
- India Trend Analysis  
- Interactive and Static Visualizations  

## 📈 Key Insights
- COVID-19 spread globally in multiple waves, peaking around early 2023  
- Death rates decreased over time due to vaccines, immunity, and better treatment  
- Global recovery rate remained above 99%  
- Asia and Europe had the highest number of cases  
- South America showed relatively higher death rates  
- India saw a major spike during the Delta wave (2021)  
- By 2024, both cases and deaths significantly decreased
